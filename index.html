Ph.D. Thesis

Title: DEEP LEARNING-DRIVEN FRAMEWORK FOR SQL INJECTION VULNERABILITY DETECTION

Objective 1: Enhancement of Naïve Based Supervised Classification

Implemented low complexity supervised classification with missing value imputation, feature selection, and ranking to achieve efficient attribute-based classification and improved accuracy.

Objective 2: Enhanced Ensemble Learning with Boosting

Developed an ensemble learning method using decision tree classifiers and forward feature selection, leveraging boosting mechanisms to improve classification accuracy.

Objective 3: Hybrid CNN-LSTM Deep Learning with BERT

Presented an autoencoder-based unsupervised mechanism integrated with CNN and LSTM modules, combined with BERT embeddings, to improve pattern recognition and classification accuracy.

Objective 4: Advanced Integrated DL Model with Attention

Designed an advanced deep learning architecture combining CNN, BiLSTM, GRU, and attention modules to enhance pattern learning capability and improve SQL injection vulnerability detection.
